Video Game Sales Project

As a data analyst for video game sales, I was tasked with cleaning up the code in Python's pandas database and using SQL to answer several key questions. These included identifying the trends of the top games in the market over the years, the most popular genres of games, and the genres and platforms with the highest sales. I also analyzed sales by country to determine which regions had the most sales.
Using the insights gained from this analysis, I created a visually appealing dashboard using Tableau that provided a user-friendly interface for exploring the data. The dashboard allowed stakeholders to easily identify trends and patterns in the data, which could be used to inform strategic decision-making.
Overall, my work on this project demonstrated my ability to work with large and complex datasets, apply advanced analytical techniques, and communicate insights effectively using data visualization tools. I believe that these skills, combined with my passion for data-driven decision-making, make me an ideal candidate for any data analyst position in the video game industry.
